To add a new component:

1. Create a folder in github-pages-hub/src/app with a name matching the component name.
2. Create 4 files in the new folder:
	- [component-name].component.css
	- [component-name].component.html
	- [component-name].component.spec.ts
	- [component-name].component.ts
3. Add the component to the App Module (app.module.ts file) to make it available to other components in the App Module.
4. Configure the app routing module (app-routing.module.ts file) with the new component.
